SETUP:
Please install the following packages if you do not have them: re, json, os, time, collections, lxml, nltk, bs4.

To install, simply enter "pip install <package>" into the terminal.

HOW TO CREATE INDEX:
Simply run index.py. It can take up to half an hour. A warning (XMLParsedAsHTMLWarning) may pop up, which can be safely ignored.

HOW TO SEARCH:
Simply run search.py and type in the terms you'd like to search. Press the ENTER key to start the search.

HOW TO STOP THE PROGRAM:
Simply terminate it like you would any other Python program/script: Ctrl+C or CMD+C
